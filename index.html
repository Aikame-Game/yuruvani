サーバー詳細情報

当サーバー「ゆるばにら」は、Minecraft Java Editionに対応したマルチプレイサーバーです。バニラサバイバルの醍醐味である「自由な開拓」を最大限に尊重するため、ゲームバランスを大きく変えるような複雑な経済プラグインやランクシステムはあえて導入しておりません。仕事終わりや学校の放課後など、日常生活の合間にほっと一息つけるような、穏やかなコミュニティ作りを目指しています。

サーバー内のルールについては、他のプレイヤーの迷惑になる行為（窃盗、破壊、不適切なチャットなど）を禁止するという、マルチプレイにおける基本的なマナーを重視しています。また、独自のバックアップシステムを完備しており、不慮のサーバーダウンや予期せぬトラブルからプレイヤーの皆様の大切な資産を保護する体制を整えています。不具合や要望がある際は、上記公式Discordまたは管理者のX（旧Twitter）までお気軽にご連絡ください。
